# Alur Presentasi Tugas Besar (5-7 Menit)

## 1. Pembukaan

“Website kami bernama **Rona Malaysia**, sebuah website edukatif yang memperkenalkan sejarah, bahasa, kuliner, busana, seni, dan arsitektur tradisional Malaysia.”

## 2. Latar Belakang

- Tema budaya dipilih karena Malaysia memiliki masyarakat multikultural dan warisan yang beragam.
- Informasi budaya sering tersebar, sehingga website menyatukannya dalam pengalaman visual dan interaktif.

## 3. Demonstrasi Halaman

1. Beranda: efek mengetik, animasi, mode gelap, fakta acak.
2. Sejarah: timeline dan pencarian.
3. Warisan: kartu budaya dan layout responsif.
4. Kuliner: filter dan random food.
5. Bahasa: dropdown frasa dan text-to-speech.
6. Galeri: filter dan modal gambar.
7. Interaktif: kuis, voting, dan validasi form.
8. Sumber: sitasi resmi.

## 4. Pemenuhan Teknis

- HTML semantik.
- CSS eksternal, mobile-first.
- Flexbox + Grid.
- Media queries.
- Transitions dan animations.
- JavaScript functions, events, conditional, looping.
- localStorage pada mode gelap dan voting.

## 5. Cross-Browser

- Layout fallback menggunakan Flexbox.
- Fitur modern seperti speech synthesis diperiksa terlebih dahulu sebelum dipakai.
- Semua fungsi utama tetap dapat digunakan tanpa framework.

## 6. Penutup

“Rona Malaysia tidak hanya menyajikan informasi, tetapi mengajak pengunjung belajar melalui eksplorasi, kuis, dan interaksi.”
