Kredit teknis

Teknologi yang digunakan

  • HTML5 semantik untuk struktur halaman.
  • CSS3 eksternal, mobile-first, flex, grid, media queries, transition, dan animation.
  • JavaScript eksternal tanpa framework untuk fungsi dan interaksi.
  • SVG lokal untuk seluruh ilustrasi; tidak mengambil template atau gambar pihak ketiga.

Catatan sitasi kode

Seluruh kode proyek ditulis khusus untuk tugas ini. Tidak ada template Bootstrap/Tailwind yang disalin. Bila proyek dikembangkan dengan library tambahan, cantumkan nama library, versi, lisensi, dan tautan dokumentasi pada halaman ini.
